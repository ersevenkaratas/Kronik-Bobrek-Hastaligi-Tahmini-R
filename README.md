# Kronik-Bobrek-Hastaligi-Tahmini-R
Projenin Amacı: Klinik ölçümler kullanarak bireyin kronik böbrek hastası olup olmadığını tahmin etmek.

Veri Seti Özellikleri: 5000 gözlem ve 11 değişken (Kreatinin, GFR, BUN vb.).

Kullanılan R paketleri: caret, e1071, class ve clusterSim paketleri.

Metot: Veriler normalizasyon işleminden geçirilmiş ve %70-%30 oranında eğitim/test olarak ayrılmıştır.

Sonuçlar: 
* KNN Başarı Oranı: 0.9953.
* Naive Bayes Başarı Oranı: 0.9813.
